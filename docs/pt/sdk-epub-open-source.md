# SDK EPUB Open Source

Um SDK EPUB open source ajuda a criar um leitor com texto reflow, navegacao e temas, mantendo controle total da UX e do roadmap. EPUB e essencial para conteudo longo e estilo livro.

## O que avaliar em um SDK EPUB

- Layout reflow com tipografia e espacamento controlados
- Sumario e navegacao por capitulos
- Busca, selecao e destaques
- Temas (claro, escuro, sepia) para conforto de leitura

## Onde o Papyrus entra

Papyrus suporta EPUB junto de PDF e TXT, usando o mesmo core e padroes de UI. No mobile, o EPUB renderiza em WebView mantendo a mesma experiencia de leitura.

Pontos chave:

- SDK EPUB open source com licenca MIT
- Estado e eventos unificados para anotacoes e busca
- UX compartilhada entre formatos (PDF, EPUB, TXT)
- UI para React e React Native no web e no mobile

## Checklist de avaliacao

- Qualidade do render EPUB e navegacao
- Fidelidade de busca e destaques
- Estrategia de WebView ou nativa
- Temas e controle de tipografia
- Documentacao e exemplos ativos
